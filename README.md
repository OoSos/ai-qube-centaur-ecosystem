## ğŸš€ **GETTING STARTED**

### **Prerequisites:**
- Node.js 18+
- Docker and Docker Compose
- Python 3.9+
- n8n Cloud or Self-hosted instance
- API access to Claude 4, OpenAI Codex, Gemini 2.5 Pro

### **Development Setup:**
```bash
# Clone and setup
cd centaur-development
npm install
pip install -r requirements.txt

# Start development environment
docker-compose up -d
npm run dev
```

### **Configuration:**
1. Copy `.env.example` to `.env`
2. Configure AI API keys
3. Setup n8n instance connection
4. Initialize vector database for RAG
